# Getting Started

ADMS RDS UI is a Vue 3 component library implementing the ASU Rocket Design System. Components are copied directly into your project, giving you full control to customize as needed.

## Installation

### 1. Configure GitHub Packages Registry

This package is hosted on GitHub Packages. You need to authenticate before installing.

**Create a GitHub Personal Access Token:**
1. Go to [GitHub Settings → Developer settings → Personal access tokens](https://github.com/settings/tokens)
2. Generate a new token (classic) with `read:packages` scope
3. Copy the token

**Configure npm to use GitHub Packages for global installs:**

For global package installations, the `.npmrc` file **must** be in your home directory. Create or edit `~/.npmrc`:

```bash
# macOS/Linux
nano ~/.npmrc

# Windows (PowerShell)
notepad $HOME\.npmrc
```

Add these lines to the file:

```bash
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_TOKEN
@banavasi:registry=https://npm.pkg.github.com
```

Replace `YOUR_GITHUB_TOKEN` with your actual token.

**Verify the configuration:**

```bash
# Check that the file exists and has correct content
cat ~/.npmrc

# You should see your token and the registry setting
```

> **Note:** Project-level `.npmrc` files are ignored for global installs. The home directory `~/.npmrc` is required.

### 2. Install the CLI

```bash
# npm
npm install -g @banavasi/adms-rds-ui-cli

# pnpm
pnpm add -g @banavasi/adms-rds-ui-cli

# yarn
yarn global add @banavasi/adms-rds-ui-cli

# bun
bun add -g @banavasi/adms-rds-ui-cli
```

### 3. Initialize Your Project

```bash
adms-rds-ui init
```

This creates the following structure:

```
your-project/
├── src/
│   ├── components/ui/          # Component directory
│   ├── lib/
│   │   └── util.ts             # Utility functions (cn, cva)
│   └── styles/                 # ASU brand styles
│       ├── _colors.scss
│       ├── _variables.scss
│       ├── _spacing.scss
│       ├── _typography.scss
│       ├── _mixins.scss
│       ├── _index.scss
│       └── styles.scss
├── vite.config.ts              # Updated with aliases
├── tsconfig.json               # Updated with paths
└── rds-ui.json                 # CLI configuration
```

### 4. Import Styles

Add to your `main.ts` or `main.js`:

```typescript
import '@/styles/styles.scss'
```

### 5. Add Components

```bash
# Add a single component
adms-rds-ui add button

# Add multiple components
adms-rds-ui add button tooltip card

# Skip confirmation prompts
adms-rds-ui add button -y
```

## Usage Example

```vue
<script setup lang="ts">
import { Button } from '@/components/ui/Button'
</script>

<template>
  <Button variant="primary">Click me</Button>
</template>
```

## Configuration

The CLI creates a `rds-ui.json` file for project-specific settings:

```json
{
  "src": "src",
  "components": "@/components/ui",
  "lib": "@/lib",
  "styles": "@/styles",
  "aliases": {
    "@/components/ui": "src/components/ui",
    "@/lib": "src/lib",
    "@/styles": "src/styles"
  }
}
```
