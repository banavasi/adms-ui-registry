# ADMS RDS UI

A Vue 3 component library implementing the **ASU Rocket Design System**.

## Features

- **Vue 3** - Built with Composition API and `<script setup>`
- **Bootstrap 5** - Uses Bootstrap utilities and components
- **ASU Brand** - Official ASU colors and design tokens
- **Accessible** - Built with Reka UI primitives for full accessibility
- **Copy-Paste** - Components live in your codebase, fully customizable

## Quick Start

### 1. Install CLI

```bash
npm install -g @banavasi/adms-rds-ui-cli
```

### 2. Initialize Project

```bash
adms-rds-ui init
```

### 3. Add Component

```bash
adms-rds-ui add button
```

### 4. Use in Your App

```vue
<script setup lang="ts">
import { Button } from '@/components/ui/Button'
</script>

<template>
  <Button variant="primary">Click me</Button>
</template>
```
